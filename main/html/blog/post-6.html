2.What is installer?

It is a program that is used to install application on the computer

3.What are the types of installation?

Attended installation, Un-attended Installation, Silent Installation, Headless Installation, Scheduled Installation, Clean Installation, Network Installation

4.What is Bootstrapper?

It is a small program designed to update the installer and start the installation automatically.

5.What is a Mobile Operating System?

Mobile Oerating System is an operating system designed for mobile devices such as phones and smartwatches

6.What are the types of mobile operating system?

There are two major mobile operating system. Android and iOS. Android is a linux-based open-source mobile operating system developed and owned by Google. iOS is a mobile OS developed by Apple for their iPhone.
